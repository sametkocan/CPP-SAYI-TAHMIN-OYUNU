# C++ Konsol Sayı Tahmin Oyunu

Visual Studio 2022 üzerinde C++ dili kullanılarak geliştirilmiş, arcade tarzında bir sayı tahmin oyunudur. Bu proje, C++ kütüphane çakışmalarını yönetmek ve döngü algoritmalarını kurmak amacıyla ilk proje olarak geliştirilmiştir.

## 🚀 Oyun Özellikleri

- **3 Farklı Zorluk Seviyesi:** Kolay (0-10), Orta (0-20) ve Zor (0-30) arası dinamik sayı aralıkları.
- **Arcade Tarzı Can Sistemi:** Toplam 5 can hakkı bulunur. Oyuncu doğru tahmin yaptığında canı azalmaz, yeni bir sayı üretilir ve puan kazanmaya devam eder.
- **Akıllı Yönlendirme:** Yanlış tahminlerde oyuncuya "Daha Büyük" veya "Daha Küçük" şeklinde ipuçları verilir.
- **Temiz Mimari:** `using namespace std;` kullanımından kaçınılarak, sadece gerekli olan `using std::cout;` ve `using std::cin;` ifadeleriyle namespace belirsizlik (ambiguous) hataları önlenmiştir.

## 🛠️ Nasıl Çalıştırılır?

1. Projeyi bilgisayarınıza indirin.
2. `Tahmin.cpp` dosyasını Visual Studio 2022 veya herhangi bir C++ derleyicisinde açın.
3. Projeyi derleyin ve çalıştırın.
